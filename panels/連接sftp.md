# 連接sftp

首先下載並安裝[FileZilla Client](https://filezilla-project.org/download.php)
![img](/img/sftp.png)

點選左上角開啟站台管理員，新增站台
![img](/img/sftp-1.png)

參照以下設定
```
協定: SFTP
主機: sftp.freeserver.tw
連接埠: 2022
登入形式: 一般
使用者: *從面板的其他設定複製
密碼: 你的面板密碼
```
